# Analiza statystyk zawodników NBA (sezon 2014/2015)  
Projekt w R

## Opis projektu
Projekt analizuje statystyki zawodników NBA z sezonu 2014/2015, ze szczególnym uwzględnieniem wpływu wzrostu oraz pozycji na kluczowe elementy gry. Analiza obejmuje testy statystyczne, modele regresji oraz wizualizacje, których celem jest wyjaśnienie różnic między zawodnikami oraz identyfikacja czynników wpływających na efektywność gry.

## Źródło danych
Dane pochodzą ze statystyk NBA (2014/2015) dostępnych publicznie m.in. w Basketball Reference.
