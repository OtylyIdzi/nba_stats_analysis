# NBA Player Stats Analysis

Analiza statystyk zawodników NBA (sezon 2014/2015) z wykorzystaniem R.  
W projekcie sprawdzono wpływ wzrostu i pozycji zawodników na kluczowe statystyki (zbiórki ofensywne, bloki, rzuty za 3 punkty, skuteczność rzutów osobistych) przy użyciu testów ANOVA i regresji liniowej/wielorakiej.
